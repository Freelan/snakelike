# snakelike
A simple snake game.

<b>Dependencies </b>(ubuntu):
<br>
<code>
	libsfml-dev
</code>
<br>
<code>
	libncurses5-dev
</code>

<b>Installation</>
<br>
<code>
	git clone https://github.com/Freelan/snakelike.git && cd snakelike
</code>
<br>
<code>
	make
</code>
<br>
<code>
	./snakelike
</code>
